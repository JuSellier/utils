# Picking the right border radius

## No radius

Sharp corners.

| Pros         |
| ------------ |
| Professional |
| Clean        |
| Organized    |

| Cons           |
| -------------- |
| Conservative   |
| Overly serious |

## 4px radius

Slightly rounded corners. Safest choice.

| Pros         |
| ------------ |
| Safe         |
| Friendly     |
| Professional |

| Cons       |
| ---------- |
| Mainstream |

## 12px radius

Rounded corners.

| Pros    |
| ------- |
| Trendy  |
| Playful |
| Young   |

| Cons      |
| --------- |
| Infantile |

## Round radius

Pill/Round shaped elements

| Pros     |
| -------- |
| Natural  |
| Creative |
| Friendly |

| Cons      |
| --------- |
| Infantile |
